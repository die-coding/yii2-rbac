Cara Memasang
-------------

Melalui console:

```
composer require diecoding/yii2-rbac "dev-master"
```

atau tambahkan:

```
"diecoding/yii2-rbac": "dev-master"
```

pada baris `require` yang terdapat di berkas `composer.json`. Kemudian jalankan

```
composer update
```
