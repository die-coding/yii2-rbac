Installation
============

Cara Memasang
-------------

Melalui console:

```
composer require --prefer-dist diecoding/yii2-rbac "@dev"
```

atau tambahkan:

```
"diecoding/yii2-rbac": "@dev"
```

pada baris `require` yang terdapat di berkas `composer.json`. Kemudian jalankan

```
composer update
```
